Cyberspace

About Cyberspace

 Source Code

Use the WASD keys to move around in the world, and the mouse to move the camera around. Click on the output screen in order to capture focus, and press esc to escape. Use the Q, E, R, T keys to alter camera speed, ascending in speed from left to right.

This project was written in C with OpenGL, and ported to the web with emscripten.

The setting is a remote island in the middle of the ocean, featuring a small outpost that includes a house, storage silo, restuarant, hangar and skyscraper. I designed all the models and world using a combination of SculptGL and OpenSCAD.
